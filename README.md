# Сравнивание вакансий программистов

Программа выводит таблицы со статистикой языков программирования, основывающейся на вакансиях на HeadHunter и SuperJob.

### Как установить

Для получения статистики с SuperJob понадобится секретный ключ, получаемый при регистрации приложения на SuperJob. Его нужно записать в файл .env: `SJ_SECRET_KEY=ключ`.

Используйте pip (или pip3, есть есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
Для получения статистики необходимо запустить файл main.py:
```
python main.py
```
